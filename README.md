# **LapyGeeks---Website**
- Laptop Shopping Website using HTML, CSS, JavaScript, Bootstrap, jQuery.
- As name suggests, its a shopping website where I have categoriezed Laptops into 3 different Categories 
  * Professionnal
  * Creator
  * Gaming
- This is only FrontEnd Website which means all the data here is static.
- There is no BackEnd provided in this website.

## **What I learned**
- Had a lot of fun and learned alot during this Project.
- Enhanced my knowledge over JavaScript, Bootstrap and jQuery.
